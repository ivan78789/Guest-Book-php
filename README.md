# Гостевая книга на PHP

Страница отображает все сообщения из таблицы `messages` и форму для добавления нового.  

- Таблица `messages`: `id`, `name`, `email`, `message`, `created_at`.  
- Валидация формы и защита от XSS через `htmlspecialchars()`.  
- Пагинация сообщений с использованием `LIMIT` и `OFFSET`.
